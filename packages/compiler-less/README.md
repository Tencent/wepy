# wepy less 编译器
# wepy less compiler


## 安装
## Install

```
npm install @wepy/compiler-less less --save-dev
```


## 配置`wepy.config.js`
## Configure `wepy.config.js`

```
module.exports = {
    "compilers": {
        less: {
            "compress": true
        }
    }
};
```
