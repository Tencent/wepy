# wepy sass 编译器
# wepy sass compiler


## 安装
## Install

```
npm install @wepy/compiler-sass --save-dev
```

## 配置`wepy.config.js`
## Configure `wepy.config.js`


```
module.exports = {
    "compilers": {
        sass: {
            'outputStyle': 'compressed'
        }
    }
};
```
