# wepy stylus 编译器
# wepy stylus compiler

## 安装 
## Install

```
npm install @wepy/compiler-stylus --save-dev
```


## 配置`wepy.config.js`
## Configure `wepy.config.js`

```
module.exports = {
    "compilers": {
        stylus: {
            "compress": true
        }
    }
};
```
