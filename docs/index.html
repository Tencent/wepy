为什么要选择WePY？
目前已有超过1000位开发者正在使用或者了解该框架开发小程序
  • 开发风格 开发风格接近于 Vue.js，支持 Vue.js 风格的组件Props 传值组件自定义事件、组件的分布式复用Mixin、计算属 性函数computed模板内容分发slot等等
  • 组件化 通过预编译手段让小程序支持组件化开发，完美的解决 了组件隔离，组件嵌套，组件通信等问题
  • NPM 支持使用第三方 npm 资源，自动处理 npm 资源之间 的依赖关系，完美兼容所有无平台依赖的 npm 资源包
  • Promise 通过polyfill让小程序完美支持Promise，解决回调烦恼
  • ES2015+ 项目以 ES2015 为基础开发，可以使用 Generator Fu- nction / Class / Async Function 等特性，大大提升开 发效率
  • 优化 对小程序本身的优化，如请求列对处理，优雅的事件处 理，生命周期的补充，性能的优化等等
  • 编译器 支持样式编译器：Less/Sass/Styus，模板编译器：wx- ml/Pug，代码编译器：Bable/Typescript。可以自由选 择熟悉的开发方式开发
  • 插件 支持多种插件处理，如文件压缩，图片压缩，内容替换 等等，扩展简单，使用方便
  • 框架大小 压缩后 24.3KB 即可拥有所有框架功能，额外增加 8.9 KB后即可使用 Promise 和 Async Function
